v 1.3 통계 레이아웃 전면 개선
🔧 개선 내 통계 — 지역별 · 종류별 · 숙성연수별 · 도수별 차트에서 바 그래프를 제거하고 항목명 + 횟수 뱃지 목록으로 변경. 항목 이름이 잘리지 않고 전체 표시됨.
🐛 수정 내 통계 — 점수 분석(향 / 맛 / 피니시 / 종합) 바 너비가 항상 100%로 표시되던 문제 수정 → 실제 점수를 100점 기준으로 표시 (83점이면 83%만 채워짐). 영문 레이블 제거.
✨ 신기능 다른 사람 통계 — 인기 위스키 Top 10으로 확장. 기본 Top 5만 노출하고 ▼ 더 보기 버튼으로 나머지 5개까지 펼칠 수 있음.
🔧 개선 다른 사람 통계 — 평균 점수 분석 섹션을 위스키 점수 랭킹으로 변경. 종합점수 높은 순으로 위스키 이름과 평균 점수를 표시 (최대 10위).

🐛 수정 평균 점수 분석 바 너비가 항상 100%로 표시되던 문제 수정 → 실제 점수에 비례해 채워지도록 개선 (83점 → 83% 표시). 레이블에서 영문 제거 (향 / 맛 / 피니시 / 종합).

v 1.1 모바일 UX 및 디자인 개선
🔧 개선 향미 프로필 슬라이더 레이아웃 변경 — 슬라이더와 레이더 차트가 나란히 배치되던 구조에서 슬라이더가 전체 너비를 사용하고 레이더 차트가 아래에 오는 세로 배치로 변경. 모바일에서 훨씬 조작하기 쉬워짐.
🔧 개선 슬라이더 터치 영역 확대 — 트랙 두께 4px → 8px, 썸 크기 16px → 28px로 확대.
🐛 수정 레이더 차트에서 스파이시 · 스모키 텍스트가 캔버스 경계에 잘리던 문제 수정 — 캔버스 크기 및 여백 재조정.
🎨 디자인 사이드바 · 하단 내비게이션 · 더보기 메뉴 · 사이드바 하단 버튼 아이콘 전체를 이모지에서 위스키 특화 SVG 아이콘으로 교체 — 위스키 병 / 글랜캐런 잔 / 바 차트 / 가격 태그 형태.
🔧 개선 사용설명서 v1.1 업데이트 — 목차 10개 → 14개 섹션 확장, 데이터 관리 섹션의 잘못된 내용(백업 불가) 수정, 향미 슬라이더 설명 반영.

v 1.0 위스키 노트 출시 🎉
✨ 신기능 위스키 컬렉션 관리 — 미개봉 · 개봉중 · 완음 상태로 분류, 사진 첨부, 구매 정보(날짜·가격·구매처) 기록, 잔여량 자동 계산.
✨ 신기능 시음 노트 — 향(Nose) · 맛(Palate) · 피니시(Finish) 텍스트 및 개별 점수 슬라이더(0~100), 액체 색깔, 시음량 기록. 시음 사진 첨부 가능.
✨ 신기능 향미 휠 레이더 차트 — 과일 · 달콤 · 스파이시 · 오크 · 스모키 · 곡물 6축 향미 프로필 입력 및 육각형 차트 시각화.
✨ 신기능 인스타그램 공유 카드 — 1080×1080px PNG 자동 생성. 위스키 정보 · 향미 휠 · 시음 노트 · 날짜 포함.
✨ 신기능 라벨 스캔 (OCR) — Google Cloud Vision API 연동. 위스키 병 라벨 사진을 찍으면 이름 · 도수 · 숙성연수 자동 입력.
✨ 신기능 증류소 자동완성 — Wikidata 기반 실시간 검색. 증류소 선택 시 지역 · 종류 자동 입력.
✨ 신기능 통계 분석 — 지역별 · 종류별 · 숙성연수별 · 도수별 · 점수 분포 차트. 취향 분석 리포트 (스타일 추천 포함).
✨ 신기능 가성비 랭킹 — 구매 가격과 시음 점수로 자동 계산. S/A/B/C/D 등급 부여. 컬렉션 카드에 뱃지 표시.
✨ 신기능 위시리스트 — 마셔보고 싶은 위스키 저장. 우선순위(🔴🟡🔵) 설정, 구매완료 처리.
✨ 신기능 데이터 내보내기 / 가져오기 — 전체 데이터(사진 포함)를 JSON 파일로 백업 및 복원. 병합 · 덮어쓰기 모드 지원.
✨ 신기능 다른 사람 통계 — 동의 사용자들의 익명 시음 데이터 집계. 인기 위스키 TOP5 · 지역별 · 종류별 분포 확인.
✨ 신기능 PWA 지원 — 홈 화면에 앱 설치 가능. 오프라인에서도 컬렉션 및 시음 노트 조회 · 작성 가능.
