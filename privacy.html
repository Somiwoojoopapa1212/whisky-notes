개인정보처리방침 · Privacy Policy

최종 업데이트: 2026년 4월 24일

위스키 노트(Whisky Notes)는 사용자의 개인정보 보호를 중요하게 생각합니다. 이 방침은 앱이 어떤 정보를 수집하고 어떻게 사용하는지 설명합니다.

핵심 요약: 위스키 노트는 이름, 연락처, 위치 등 어떠한 개인 식별 정보도 수집하지 않습니다. 모든 시음 기록은 사용자의 기기에만 저장됩니다.

1. 수집하는 정보

위스키 노트는 다음 정보를 처리합니다:

  • 기기 내 저장 데이터 — 위스키 컬렉션, 시음 노트, 위시리스트, 사진 등 사용자가 직접 입력한 데이터. 기기 로컬(localStorage, IndexedDB)에만 저장되며 외부로 전송되지 않습니다.
  • 익명 통계 데이터 (선택 동의 시) — 앱 첫 실행 시 '데이터 공유 동의'에 동의한 경우에 한해, 시음 날짜·위스키 종류·지역·점수 등의 익명 데이터가 Firebase Firestore에 저장되어 커뮤니티 통계에 활용됩니다. 이름, 연락처, 위치 등 개인 식별 정보는 포함되지 않습니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 기기 위치(GPS)
  • 연락처, 사진 앨범 접근 (사용자가 직접 선택한 사진만 처리)
  • 광고 식별자

3. 정보 사용 목적

  • 앱 내 시음 기록·컬렉션 관리 기능 제공
  • 동의 사용자의 익명 데이터를 집계하여 커뮤니티 통계(다른 사람 통계) 기능 제공

4. 제3자 서비스

위스키 노트는 다음 제3자 서비스를 사용합니다:

  • Google Firebase Firestore — 동의 사용자의 익명 시음 통계 저장. Google의 개인정보처리방침이 적용됩니다.
  • Wikidata API — 증류소 이름 자동완성 기능. 검색어만 전송되며 개인 정보는 포함되지 않습니다.

광고 네트워크, 분석 도구(Google Analytics 등)는 일절 사용하지 않습니다.

5. 데이터 보관 및 삭제

  • 기기 로컬 데이터는 앱 삭제 또는 앱 내 '초기화하기' 메뉴로 언제든지 삭제할 수 있습니다.
  • 익명 공유 데이터는 Firebase에 저장되며, 삭제를 원하실 경우 아래 이메일로 문의해 주세요.

6. 아동 개인정보 보호

위스키 노트는 주류 관련 앱으로 만 19세 이상을 대상으로 합니다. 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 방침 변경

본 방침은 변경될 수 있으며, 변경 시 이 페이지를 통해 공지합니다.
