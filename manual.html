앱 소개 및 주요 기능

위스키 노트는 위스키 애호가를 위한 개인 컬렉션 관리 및 시음 기록 앱입니다. 별도의 회원가입 없이 스마트폰에 바로 설치하여 사용할 수 있으며, 모든 데이터는 내 기기에 안전하게 저장됩니다.

📦
컬렉션 관리
보유한 위스키를 미개봉 · 개봉중 · 완음 상태로 관리. 구매 정보 · 사진 · 잔여량 추적.
📝
시음 노트
향 · 맛 · 피니시를 텍스트로 기록. 세부 점수 슬라이더, 시음 사진 첨부 지원.
🎯
향미 휠 (레이더 차트)
과일 · 달콤 · 스파이시 · 오크 · 스모키 · 곡물 6축으로 향미 프로파일 시각화.
📤
인스타 공유 카드
1080×1080 PNG 카드 자동 생성. 위스키 정보 · 향미 휠 · 시음 노트 포함.
📷
라벨 스캔 (OCR)
병 라벨 사진으로 이름 · 도수 · 숙성연수 자동 인식. Google Vision API 사용.
🔍
증류소 자동완성
증류소 입력 시 지역 · 종류 자동완성. Wikidata 기반 실시간 검색.
📊
통계 분석
지역별 · 종류별 · 숙성연수별 · 도수별 시음 통계 차트.
🌐
커뮤니티 통계
동의한 사용자들의 익명 시음 데이터 집계. 인기 위스키 TOP 5 확인.
💡 오프라인 사용 가능: 앱 설치 후에는 인터넷 없이도 컬렉션 조회 및 시음 노트 작성이 가능합니다. 증류소 자동완성과 커뮤니티 통계는 인터넷 연결 시에만 작동합니다.

설치 방법

위스키 노트는 PWA(Progressive Web App) 방식으로 배포됩니다. 앱 스토어 설치 없이 브라우저에서 직접 설치하며, 일반 앱처럼 홈 화면에서 실행됩니다.

https://somiwoojoopapa1212.github.io/whisky-notes/

Android 안드로이드 설치

  • Chrome 브라우저를 열고 위 주소로 접속합니다.
  • 화면 하단 또는 상단에 앱 설치 배너가 나타나면 탭합니다.
    배너가 없으면 다음 단계로 이동합니다.
  • 배너가 없을 경우: 오른쪽 상단 점 3개(⋮) 메뉴 → 홈 화면에 추가 또는 앱 설치를 선택합니다.
  • 확인 팝업에서 설치를 탭합니다.
  • 홈 화면에 위스키 노트 🥃 아이콘이 추가됩니다.
📌 앱 내 설치 버튼: 처음 접속 시 앱 상단에 📲 앱 설치 버튼이 나타날 수 있습니다. 이 버튼을 탭하면 동일하게 설치됩니다.

iOS 아이폰 / 아이패드 설치

⚠️ 주의: iOS에서는 반드시 Safari 브라우저로 접속해야 합니다. Chrome · 네이버 앱 등 다른 브라우저에서는 홈 화면 추가가 지원되지 않습니다.
  • Safari를 열고 위 주소로 접속합니다.
  • 하단 메뉴바의 공유 버튼 ⬆️을 탭합니다.
    (아이패드는 주소창 옆 공유 아이콘)
  • 스크롤을 내려 홈 화면에 추가를 탭합니다.
  • 앱 이름을 확인 후 오른쪽 상단 추가를 탭합니다.
  • 홈 화면에 위스키 노트 🥃 아이콘이 추가됩니다.
💡 iOS 팁:
  • 홈 화면에서 실행 시 주소창이 없는 전체화면 앱 모드로 동작합니다.
  • iOS 16.4 이상에서는 홈 화면 앱 아이콘 꾹 누르기 → 편집으로 아이콘 이름 변경 가능.

사진 등록 (크롭)

사진 영역을 탭해 갤러리에서 이미지를 선택하면 정사각형 크롭 화면이 나타납니다.

  • 드래그로 위치 조정
  • 슬라이더로 확대/축소
  • 확인 탭으로 저장

카드 레이아웃

등록된 위스키는 왼쪽 정사각형 사진 + 오른쪽 정보 형태의 카드로 표시됩니다.

  • 상태 뱃지 (미개봉/개봉중/완음)
  • 시음 횟수 · 잔여량 (개봉중일 때)
  • 구매 가격

상태 필터

컬렉션 탭 상단의 전체 미개봉 개봉중 완음 버튼으로 필터링할 수 있습니다.

증류소 자동완성

위스키 추가/수정 모달에서 증류소 필드에 2글자 이상 입력하면 Wikidata 데이터베이스를 검색하여 실제 증류소 목록을 보여줍니다.

  • 증류소 필드에 증류소명 입력 (예: Glenfarclas, 아드벡)
  • 0.4초 후 드롭다운 목록이 자동으로 나타납니다.
  • 목록에서 원하는 증류소를 탭합니다.
  • 지역과 종류가 자동으로 입력됩니다.
💡 자동 입력 예시:
Glenfarclas → 지역: 스페이사이드 · 종류: 싱글몰트 자동 입력
Ardbeg → 지역: 아일라 · 종류: 싱글몰트 자동 입력

이미 지역이나 종류가 입력된 경우에는 덮어쓰지 않으며, 비어있는 항목만 자동 채웁니다.

📡 인터넷 연결 필요: 자동완성은 Wikidata API를 실시간으로 호출하므로 인터넷 연결이 필요합니다.

🎯 향미 휠 (Flavor Wheel)

6개의 향미 슬라이더로 위스키의 향미 프로파일을 입력하면 실시간으로 레이더 차트 미리보기가 업데이트됩니다.

🍎
과일 (Fruity)
사과 · 배 · 복숭아 · 체리 등 과일향
🍯
달콤 (Sweet)
바닐라 · 꿀 · 캐러멜 · 초콜릿
🌶️
스파이시 (Spicy)
후추 · 생강 · 계피 · 넛멕
🌳
오크 (Oaky)
오크 · 우디 · 탄닌 · 셰리
🔥
스모키 (Smoky)
피트 · 훈연 · 재 · 해초
🌾
곡물 (Grainy)
보리 · 밀 · 빵 · 시리얼

각 슬라이더는 0~5 단계로 설정합니다. 0은 해당 향미 없음, 5는 매우 강함을 의미합니다.

📤 인스타그램 공유 카드 생성

시음 노트 카드의 📤 버튼을 탭하면 1080×1080px PNG 파일이 자동 생성되어 다운로드됩니다.

카드에 포함되는 정보:

  • 위스키 이름 · 증류소 · 지역
  • 종합 점수
  • 시음 사진 (있을 경우)
  • 향미 레이더 차트 (향미 데이터가 있을 경우)
  • 향 · 맛 · 피니시 노트
  • 기록 날짜 · 앱 워터마크
💡 인스타그램에 공유하려면: 다운로드된 PNG 파일을 갤러리에서 찾아 인스타그램 스토리 또는 피드에 업로드하세요.

제공 차트

  • 지역별 분포 — 스페이사이드 · 하이랜드 · 아일라 등
  • 종류별 분포 — 싱글몰트 · 블렌디드 · 버번 등
  • 숙성연수별 — NAS · 10년 이하 · 11~15년 · 16~20년 · 21년 이상
  • 도수별 — 40% 미만 · 40~45% · 45~50% · 50~55% · 55% 이상
  • 점수 분석 — 향 · 맛 · 피니시 · 종합 평균 점수 비교

요약 카드에서는 총 시음 횟수 · 총 시음량(ml) · 위스키 종류 수 · 평균 점수를 한눈에 확인할 수 있습니다.

취향 유형

  • 균형 잡힌 취향 — 최고-최저 점수 차이가 0.8 미만으로 모든 향미를 고루 즐김
  • 복합적 취향 — 상위 1·2위 향미 점수가 근접(85% 이상)하여 두 향미를 동시에 선호
  • 뚜렷한 취향 — 특정 향미가 압도적으로 높아 명확한 선호 스타일이 있음

💡 활용 팁: 마트 · 면세점 · 바에서 발견한 위스키를 그 자리에서 바로 위시리스트에 추가해두면 나중에 구매 결정에 도움이 됩니다.

가성비 랭킹

통계 탭 하단 💰 가성비 랭킹 섹션에서 내 위스키 컬렉션의 가성비를 자동으로 분석하고 순위를 확인할 수 있습니다.

📋 표시 조건: 컬렉션에 구매가격이 입력되어 있고, 해당 위스키에 종합 점수가 기록된 시음 노트가 1개 이상 있어야 랭킹에 표시됩니다.

가성비 지수 계산 방식

가성비 지수 = 평균 시음 점수 ÷ (구매가격 / 10,000)
예: 평균 80점 / ₩100,000 → 지수 8.0 (S급)

같은 가격이라면 점수가 높을수록, 같은 점수라면 가격이 낮을수록 높은 가성비 지수를 얻습니다.

가성비 등급 기준

등급가성비 지수예시
S급8.0 이상80점 / ₩100,000 → 지수 8.0
A급5.0 ~ 8.075점 / ₩150,000 → 지수 5.0
B급3.0 ~ 5.075점 / ₩250,000 → 지수 3.0
C급2.0 ~ 3.075점 / ₩375,000 → 지수 2.0
D급2.0 미만75점 / ₩500,000+ → 지수 1.5

컬렉션 카드 가성비 배지

컬렉션 페이지의 위스키 카드에도 가성비 등급 배지(💰S, 💰A 등)가 표시됩니다. 배지 위에 커서를 올리면 상세 지수를 확인할 수 있습니다.

💡 활용 팁: 가성비 지수는 절대적인 품질 지표가 아닙니다. 같은 위스키도 구매처에 따라 가격 차이가 있으므로, 참고 지표로 활용하세요.

커뮤니티 통계

더보기 메뉴(⋯)에서 🌐 커뮤니티 통계를 탭하면 다른 사용자들의 익명 시음 데이터를 확인할 수 있습니다.

  • 지역별 · 종류별 분포

Google Cloud Vision API 설정

라벨 스캔 기능은 Google Cloud Vision API를 사용합니다. 사용자 각자가 본인 계정으로 API 키를 발급받아 앱에 등록해야 합니다.

🔒 중요: API 키는 내 기기에만 저장되며 다른 사람과 공유되지 않습니다. 타인의 API 키를 사용하거나 공유하지 마세요.

무료 사용량

항목내용
월 무료 요청 수1,000건/월
초과 요금1,000건당 약 $1.50 (라벨 스캔 기준)
결제 등록 없이무료 $300 크레딧 90일 제공 (신규)

월 1,000번이면 매일 33번 스캔 가능 — 개인 사용자는 무료 범위에서 충분히 사용 가능합니다.

API 키 발급 방법

  • Google Cloud Console 접속
    https://console.cloud.google.com/
  • Google 계정으로 로그인 후, 상단 프로젝트 선택 → 새 프로젝트를 클릭합니다.
    프로젝트 이름 예시: whisky-notes
  • 왼쪽 메뉴 API 및 서비스 → 라이브러리로 이동합니다.
  • 검색창에 Cloud Vision API를 검색 후 클릭 → 사용 설정을 클릭합니다.
  • 왼쪽 메뉴 사용자 인증 정보 → 상단 + 사용자 인증 정보 만들기 → API 키를 선택합니다.
  • 생성된 API 키를 복사합니다.
    예: AIzaSyA···············
  • (권장) 키 제한을 클릭 → API 제한에서 Cloud Vision API만 선택 후 저장합니다.

앱에 API 키 등록

  • 앱 하단 더보기 ⋯ 메뉴 → ⚙️ 설정을 탭합니다.
  • Vision API 키 입력란에 복사한 키를 붙여넣습니다.
  • 저장을 탭합니다.
  • 이후 위스키 추가 화면에서 📷 라벨 스캔을 사용할 수 있습니다.
💡 라벨 스캔 팁:
  • 라벨이 선명하게 보이는 사진일수록 인식률이 높습니다.
  • 빛 반사나 그림자가 없는 평평한 배경에서 촬영하세요.
  • 인식 결과는 참고용으로 확인 후 수정이 가능합니다.

자주 묻는 질문 (FAQ)
Q. 앱을 재설치하면 데이터가 사라지나요?

A. 네, 현재 버전은 기기 로컬에 저장하므로 앱 삭제 시 데이터도 함께 삭제됩니다. 브라우저 데이터 삭제(캐시 초기화) 시에도 동일합니다.

Q. 여러 기기에서 동기화할 수 있나요?

A. 현재 버전은 단일 기기 전용입니다. 기기간 동기화는 향후 업데이트 예정입니다.

Q. 라벨 스캔이 인식을 못할 때 어떻게 하나요?

A. 라벨이 선명하게 보이도록 조명을 확보하고, 최대한 정면에서 촬영해 보세요. 인식 결과가 없거나 부정확하면 수동으로 입력하시면 됩니다.

Q. Vision API 키는 다른 사람과 공유해도 되나요?

A. 공유하지 않는 것을 강력히 권장합니다. 다른 사람이 내 키를 사용하면 무료 한도가 빠르게 소진되고, 예상치 못한 과금이 발생할 수 있습니다. 각자 본인 계정으로 발급받으세요.

Q. 바나 레스토랑에서 마신 위스키도 기록할 수 있나요?

A. 네. 시음 노트 추가 시 위스키 선택에서 ✏️ 직접 입력 (바 / 레스토랑 등)을 선택하면 컬렉션에 없는 위스키도 자유롭게 기록할 수 있습니다.

Q. 향미 휠을 입력하지 않아도 되나요?

A. 네, 향미 슬라이더는 선택 항목입니다. 모두 0으로 두면 향미 차트는 표시되지 않습니다.

Q. 피드백이나 버그는 어디에 제보하나요?

A. 더보기(⋯) → 피드백 보내기에서 이메일 또는 인스타그램 DM으로 제보할 수 있습니다.
